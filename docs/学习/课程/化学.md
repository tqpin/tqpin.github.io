比较微观粒子和宏观物体的特征：
1. 宏观物体同时具有确定的坐标和动量，可用牛顿力学描述；而微观粒子没有同时确  
定的坐标和动量，需要用量子力学描述。  
2. 宏观物体有连续可预测的运动轨道，可追踪个物体的运动轨迹加以分辨；微观粒子  
具有概率分布的特性，不可能分辨出各个粒子的轨迹。  
3. 宏观物体可以处于任意能量状态，体系的能量可以是任意的，连续变化的数值；微  
观粒子只能处于某些确定的能量状态，能量的改变量不能取任意值的，连续变化的  
数值，只能是分立的，即量子化的。  
4. Indeterminacy对宏观物体无实际意义，Planck常数可当作0；微观粒子遵循  
Indeterminacy 关系，Planck常数不能被看作0. 所以可以用不确定关系式作为区分
