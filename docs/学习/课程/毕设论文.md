# 0 摘要

基于模型的人体姿态估计目前有两种不同的方法。基于优化的方法以迭代的方式将参数体模型与二维观测相匹配，导致精确的图像模型对齐，但通常对初始化缓慢和敏感。相反，基于回归的方法，使用深度网络直接从像素估计模型参数，往往提供合理的，但像素不准确，但需要大量的监督。在这项工作中，我们不是研究哪种方法更好，而是认为这两种范式可以形成强有力的合作。一个合理的，直接从网络回归估计可以初始化迭代优化，使拟合更快，更准确。类似地，迭代优化的像素精确拟合可以作为网络的强有力监督。这是我们提出的方法 SPIN (SMPL oPtimization IN the loop) 的核心。深度网络初始化一个迭代优化程序，将身体模型拟合到训练循环内的 2D 关节，然后使用拟合估计监督网络。我们的方法本质上是自我改进的，因为更好的网络估计可以引导优化到更好的解决方案，而更准确的优化拟合可以为网络提供更好的监督。我们展示了我们的方法在不同环境下的有效性，其中 3D 地面真相是稀缺的，或不可用的，我们始终比最先进的基于模型的姿态估计方法的显著优势。

# 1 绪论

三维人体重建任务主要是希望从包含人体的一幅图像、几个角度的图像或者一系列视频中估计人体关节点位置以及人体高矮胖瘦等形态信息。在本节中，本文首先介绍了三维人体重建的大量潜在应用，以突出本课题研究的意义，然后讨论了三维人体重建的几种研究方向的分类，最后描述了本文的主要研究方向。

## 1.1 应用前景
因为三维人体模型包含了人体的姿态和形态信息，并且比二维的姿态估计提供了更多的深度信息，所以三维人体重建有更多更广泛的应用前景。这里我们提供了一些有趣的应用方向：
* VR/AR：虽然以视频为基础的内容共享系统极大的帮助了人们直接目睹事件的发生，或者替代面对面会议之类的情况，但这些方式依然缺少物理的真实的共享感。而比如谷歌的 JUMP 和微软的 Holoportation 可以提供共享的 3D 呈现，可是面临着所需设备价格昂贵的问题，迟迟无法让普通用户直接得到 3D 图像。如果可以用简单的设备重建 3D 人体，将会极大的推进内容共享系统的创新。 [Google Scholar](https://scholar.google.com/scholar_lookup?title=Microsoft%20kinect%20sensor%20and%20its%20effect&publication_year=2012&author=Z.%20Zhang)
* 电子商务的时尚领域：在过去的十年之中，消费者行为主要从传统的现场购物转向在线电子购物一直是主要趋势，随着 2020 年冠状病毒大流行，电子商务占所有服装销售的比例有了更大幅度的增加，市场对在线换衣等需求越来越大。深度相机、传感器等方法可以精准的重建人体表面的集合形状，但条件苛刻，难以推广；基于模型的方法能够很方便的恢复形态和姿势，但精度对于重建衣服表面纹理依然不够。[论文](https://ieeexplore.ieee.org/abstract/document/9483564) 
* 医疗健康：对于医疗应用来说，三维人体建模可以帮助医生获取病人的体脂率、追踪体型变化、分析病人行为姿态等等，但相对来说需要更加高的准确性和可靠性，以及会有需要长时间追踪等等额外需求。现在为医疗机构设计的系统通常价格昂贵、不灵活，需要训练有素的专业人员操作，公众很难使用高水平的医疗级扫描系统对自己进行扫描并获得健康数据反馈。研究人员正在寻求轻量化的、经济高效的三维人体重建解决方案。[link](https://icg.gwu.edu/accurate-3d-human-body-reconstruction-system-medical-purpose) 
* 体育运动：三维模拟和数字技术已经被广泛的应用到运动训练中，而运动教练在三维重建的基础上，可以进一步分析运动员的运动姿态和运动特点。比如通过人体运动跟踪，可以分析运动步长，建立整体运动轨迹；通过人体三维模型，我们可以分析人体各个部位的运动，包括平移和旋转自由度等等。从而更好的指导运动员的训练。[link](https://www.researchgate.net/publication/314653591_Study_on_the_Movement_Posture_and_Sports_Characteristics_Based_on_the_3D_Reconstruction_of_Human_Body) 
* 视频监控与行为分析：基于视频监控的人的行为分析已经得到了广泛的研究。但是作为一种消费视频应用，自动监控需要在有足够高的精度的同时，满足其实时性和轻量化。并且在应用上来看，我们需要分析人的运动和姿势，并以此为重要线索了解人物活动和理解场景。研究者希望能在此基础上快速识别检索入室盗窃、老年人跌倒等特殊行为的视频序列。[link](https://scholar.google.com/scholar?q=Automatic+Video-Based+Human+Motion+Analyzer+for+Consumer+Surveillance+System&hl=en&as_sdt=0&as_vis=1&oi=scholart) 
三维人体重建还有其他许多应用，是计算机视觉中一个基础的研究领域，并且有着活跃的研究氛围，对人们生活生产都有着重要的意义。

## 1.2 三维人体重建的研究现状

三维人体重建领域包括多种不同的背景设置，在本节中我们简单的描述了从输入数据类型，包含人体数量以及交互方式，以及重建结果三个方面上的分类，并且简单描述了各个设置的优缺点。

（1）从输入数据上分类
从三维重建基于的输入类型来看，该领域的研究方向非常多。除了传统的 RGB 图像或者视频，还包括具有深度信息的设备拍摄的图像或视频、点云数据、雷达数据、其他传感器（比如加速度仪或者分布在鞋底的压力传感器）数据等等。传统的 RGB 数据获取难度较小，设备成本也较低，因此我们主要考虑这种方法。

从 RGB 图像得到人体三维模型也分成两种类型，分别是从单一视角的单张图片，和从几个视角的多张图片两种重建模型的方式。从单一视角的图片得到模型的问题本质上是一个不适定的问题，因为很难从一维的、不包含深度信息的图像数据得到一个完整的三维人体模型，因此该类型的研究大多需要依赖于人体结构与骨架、二维关节点和轮廓等先验信息，并且主要基于特定人体模型，比如 SMPL，得到一个参数化的人体重建结果。而从多个视角的图像重建虽然发展更早，而且重建精度更高，能够得到更准确的三维人体模型，但也面临这许多问题，比如摄像系统的标定过程的准确性和鲁棒性，又比如多视图系统往往价格昂贵，且部署不易，很难被普通用户广泛接受。而在本文中，我们的任务是基于单一视角的单张图片的。

（2）从包含人物等交互上分类
三维人体重建还包括针对单个人的重建，以及考虑多个人或者人和物体之间相互交互的场景重建。对于单人重建来说，我们只需要考虑自遮挡问题，但对于人群或者人物交互的重建任务来说，我们还需要考虑人与人之间、人与物之间的遮挡问题，以及很多不一致的深度问题。对于多人重建来说，典型的方法分为自下而上和自上而下的两种类型。对于自下而上来说，我们需要检测图中所有身体关节，然后将其分组，即找到对应的人物组别。但这种方法只能较好的处理关节姿态，很难扩展到形状估计的问题上。对于自上而下的方法，我们首先需要检测场景中所有的人，然后对每个人物进行重建。而单一人体的重建任务是其他的基础，所以在本文之中，我们只考虑单人的重建问题。

（3）从重建结果上分类
最后从方法上，或者重建的结果和需求上来看，主要有两种类型，分别是基于模型的和无模型的。无模型就是力求完整的还原人体表面皮肤或者衣服的纹理，用比较高的精度将人体数字化。这种方法往往依赖于多视角系统，需要严格的环境约束 (比如大量的相机、受控照明) ，成本很高，部署也很难。而近几年研究人员试图利用从粗到细的多层次框架对高分辨率的图像进行端到端的学习，从而从单一视角的图像中恢复出数字化的人体模型 [pifu](https://scholar.google.com/scholar?hl=en&as_sdt=0%2C5&q=pifu&btnG=) 。

但这些方法还是很难取得较高的精度，因为人体是一个复杂的形状，并且能动态的具有多种不同的变化，并且人体表面的细节信息很难从单一图像中恢复出来。因此另一条路线，即基于模型的路线的主要思想就是为了减少训练参数，缩小模型的状态空间，我们可以用统计学的方式找到一种能够用较少参数简单的描述人体形态的模型，然后让系统基于特定模型估计输入中的人体姿态。虽然这样的模型大多缺少衣服等外观细节，但对于面部表情、手部动作等细节也可以参数化的进行表示，并且极大的方便了系统的学习，甚至做到实时，因此可以满足很大一部分的现实要求。

总而言之，三维人体重建是计算机视觉领域一个基础且活跃的研究方向。随着数据集的不断扩张以及计算机深度学习能力的提升，三维人体重建也不断取得着巨大的进展。在本文当中，我们的主要任务是基于单一视角的 RGB 图像，对单个人体做基于模型的三维重建。


# 2 研究背景

为了实现从单张 RGB 图像中重建基于参数模型的三维人体，目前有两种主要的方式，分别是基于优化的方法和基于回归的方法。本节主要介绍 SMPL 模型的基本信息，包括参数含义以及如何从参数转化成蒙皮，以及优化方法和回归方法的大致情况。

## 2.1 SMPL 模型

SMPL 模型，全称 Skinned Multi-Person Linear Model，是目前最流行最成功的三维人体参数模型。它基于蒙皮（skinning）以及参数渲染形状（blend shapes）来描述人体姿态模型，并从数千个 3D 身体扫描中学习得到这些统计参数。SMPL 的宗旨是使人体模型尽可能简单和标准，能够用简易的公式进行描述。更重要的是，凭借简单的顶点拓扑结构，SMPL 模型可以使得计算机视觉和动画研究者以极高的渲染效率和学习能力，逼真的表示各种人体形状。并且它具有完善的工具支持，能够与现有的渲染引擎兼容。

### 2.1.1 SMPL 模型的定义参数

SMPL 模型将人体形态分解为每个人体不同的形状参数和非刚性的、不依赖于特定人体的姿态参数。然后基于顶点的蒙皮方法，渲染出一个完整的人体模型。
（1）标准网格 T
标准网格是 SMPL 模型用来描述人体形态的三维拓扑结构，不依赖于特定人体或者姿势。标准姿态通常以形状像英文字母 T 的姿态展示，因此又被称为 T-pose。SMPL 模型的网格包含 N=6890 个顶点。
（1）形状参数 β
形状参数是用来描述特定人体的网格顶点相对于标准网格 T 的偏移量的，这些偏移可以用来描述包括高矮、胖瘦、腰围以及头身比例等等的形状。SMPL 模型把它视为一个向量，用 beta 表示。SMPL 模型包含多达三百个主成分，但前十个主成分可以概括大部分形态变化，因此我们通常使用十个维度的形状参数。使用更多的参数可以让我们对特定身体特征有更细粒度的控制，比如皮肤纹理和面部特征变化等等。
（2）姿态参数 theta
姿态参数也可以用来描述人体的网格顶点相对于标准网格 T 的偏移量，这里的偏移与特定人体形状无关，但与人体姿势有关。SMPL 通常用 23 个的旋转矩阵来表示 24 个人体关节的姿势变化，每一个旋转矩阵表示相应关节对应父节点的相对旋转。这里的数量并不和人体关节点一一对应，并且不同的数据集对关节点的回归位置有着不同的定义，比如 COCO 的数据集就把髋关节的位置放在靠外侧的位置，而 SMPL 则放在旋转的位置上。

## 2.2 研究方法

虽然从单张图片恢复人体三维结构是一个非正定的问题，但是得益于人体参数模型对人体描述的统计学简化，我们可以用较少的参数控制生成一个比较真实的人体模型。大部分基于模型的方法可以分为三类：基于优化的方法，基于学习的方法以及结合两种方式的方法。本节主要介绍了这三类方法，并且简单描述了优缺点。

### 2.2.1 基于优化的方法

基于优化的方法主要通过迭代拟合的过程，对统计学模型的参数进行调整，以减少设定的损失函数。损失项通常包括估计值与观测值的关节点位置差距、不合理的骨架弯曲。甚至优化的方法可以利用一些人体模型的先验知识，比如在 SMPLify-x 中，作者提出了一种变分自动编码器 VPoser，可以学习人类姿势的潜在表示，并将潜在代码的分布拟合成为正态分布，以作为人体姿态模型的先验知识，帮助图像和模型的拟合。[link](https://github.com/nghorbani/human_body_prior) 

优化的方法往往能够更好的收敛，并且不需要依赖于庞大的数据集，而且能够利用一些关于人体模型的先验知识。但优化的缺点也很显著，比如它对初始值非常敏感，并且对于每张图片都需要很长的时间来处理，在[5, 17, 19, 24, 25, 39SCOPE]中，它们需要几十秒来适应三维人体姿态。在早期数据集还不够的时候，优化的方法能取得很好的结果，但是当数据集越来越丰富，优化的速度劣势就体现了出来。因此，优化的方法还是无法满足在实时之类的工业领域的应用。

### 2.2.2 基于学习的方法

基于学习的方法通常先利用先进的关节点检测技术从图像中学到关于关节点的热图，然后让图片通过神经网络得到压缩后的特征图，最后两者结合在一起直接对人体统计学模型的参数进行回归。由于数据集的扩大，这种方法后续有很多的扩展和讨论，比如利用日益精准的 3d 骨骼预测直接从图像学习 SMPL 模型的两个参数，又利用 GCN 或者 transformer 直接学习 SMPL 模型的所有顶点。而在 2020 年的时候 Pose2Pose 意识到 SMPL 模型的旋转参数可以分解成扭转和旋转两方面，其中旋转可以直接利用 3D 的骨架信息得到，而扭转则更依赖于图像的视觉信息。然后在 2021 年的时候 HybrIK 利用这个分解，从图像中直接学习形状和扭转，然后从 3D 骨架中得到旋转，通过一个逆向动力学的框架，最终得到三维人体姿态模型所需要的几个参数。 
与基于优化的方法相反的是，基于学习的方法极大的加快了速度，但是这种方法也存在一些缺点，比如很难利用模型的先验知识，并且对于数据集的数量要求很高。目前并没有一个完整的庞大的数据集可以支持模型的训练和测试，因此在实践过程中，研究人员通常把多个数据集合并起来用于训练和测试，因此在测试阶段可能会存在不同数据集之间的域差（domain gap）。

### 2.2.3 混合方法

因为学习和优化的方式都有各自的优缺点，因此存在另一种方法，他们试图结合两者的优点，来完成人体三维姿态模型的估计。比较有名的一种方法是 SPIN，它首先利用一个神经网络来回归三维人体模型的几个参数，然后把这些回归得到的数值作为优化的方法的初始值，然后再通过优化的框架迭代。然后让这些参数再次进入神经网络，进行下一次的循环。
SPIN 就是这样通过这种把一个结果最为另一个的初始值的方式，来结合两者的优点，但是这样的方式并不紧密，更像是两个独立的模型。并且随着数据集的增加，优化能够利用先验知识来约束模型的优势减弱了，直接利用大量数据集也可以训练出很好的模型，因此这种混合的方法后续很少发展。

# 3 研究方法

在调查了优化、学习以及混合的方法的发展之后，我们发现优化的方法在 SMPLify 之后很少有发展，只有在生成和预测下一帧的人体模型的任务中，对构建人体模型先验的方法有所改进。而因为数据集的扩大以及三维人体姿态标注的增加，学习的方法有了长足的发展。在很多用学习的方法进行训练的论文中，大多倾向于直接用神经网络从图中学习优化方法的其他约束项，比如骨骼长度、旋转角度等。但是如果状态空间已经被学习的方法优化到了一个较小的范围，那么优化的方法更加容易收敛到最佳值的优势就体现了出来。在一个较小的状态空间中，优化的方法依然可以迅速的调整模型参数，收敛到更好的结果上。
