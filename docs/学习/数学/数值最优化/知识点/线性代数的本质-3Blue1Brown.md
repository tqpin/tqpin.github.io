## 01 02 vector，basis，span

* 三种观点看待向量
	* 方向，物理学意义：像一个箭头，没有固定的起始点
	* 列表，计算机意义：就是一组数字，可能每行都表示不同的意义，有不同的单位
	* 数学意义：**试图统一上述两种**，所以首先构建一个**坐标系**，表示不同维度有不同含义，其次规定向量起始点在原点，终点在**向量坐标**上，于是就有了一个“箭头”
	* 在运动的时候，虽然考虑的是坐标点，但可以视为一个从原点出发的向量
	* 把向量视为一个点，但注意每个点都落在空间的网格中，对向量操作和对空间操作是等价的
* 基
	* 向量空间的基是张成该空间的一个线性无关向量集
	* 每一个线性无关的基都会增加张成空间的一个维度
	* 把基看做空间的锚点
* 乘法 - 标量
	* 标量可以对基进行缩放，可以用这种观点来看待很多乘法
* 加法
	* 就是对每个维度相加

## 03 04 矩阵，线性变换，复合变换

* 线性变换
	* linear transform 中的 transform 其实表示一种函数的变化的观点，也提示了应该从运动的角度思考线性变换后空间的变换
	* 线性变换：保持网格平行且等距分布的变换；非线性变换后本来的直线可能会变成曲线
* 那么如何用数字来代表空间的形变？变化后的空间可以用变化后的基表示 
	* ![[Pasted image 20220515154040.png]]
* 矩阵的视角
	* 矩阵：看做用数字来描述空间的变换
		* 比如，一个二维线性变换由四个数字完全确定，构成一个 2x2 矩阵
	* 矩阵向量乘法：可以描述某个向量在空间变换后的位置
		* 例子：左乘矩阵拉伸、剪切、降维（列向量线性相关）
* 空间复合变换
	* 一组矩阵连乘 = 多个操作连续作用在向量/基/空间上，从左往右，像 $f (g (x))$ 一样读
	* 练习：考虑矩阵乘法的意义；考虑更高维度的意义

## 05 行列式

* 一个矩阵的行列式：表示空间经过该矩阵变换后，区域面积的放缩比例
* 负数：与空间定向有关
	* 比如二维空间，像纸一样翻转
	* 或者考虑基向量的顺序，二维空间翻转后，j 在 i 的右侧
	* 考虑 det = 0 时，i&j 重合了，空间被压缩了，然后 det < 0 时，j 就跑到右侧去了

## 06 逆矩阵，列空间，秩，零空间

* 考虑线性方程组，且正方形矩阵
* 行列式不等于零时
	* Ax=b 就是寻找一个 x，使得在经过 A 表示的空间变换后，能落在 b 上
	* 逆矩阵可以帮助让 b 落在需要求的 x 上
* 行列式等于零时
	* 空间维度被降低了，因此有可能无解
	* rank，秩：表示经过矩阵 A 变换后的空间维度
* 列空间：经过矩阵 A 变换后所有可能的结果集合，会构成一个空间
	* 即列张成的空间，即新的基的空间
	* 维度与秩相同
	* 零向量必然在列空间中
* 零空间，核：变换后落在零点的所有结果集合
* ==非方阵==：不同空间

## 07 点积，对偶性

* 点积的理解
	* 数值计算：向量点积 = 每个维度相乘，再相加
	* 几何解释：把某个向量投影到另一个向量，再相乘
* 为什么点积顺序可互换，但投影似乎不是一个可换的过程？
	* 把两个向量放缩成等长之后，投影就是可互换的过程了，然后在乘以 scalar 就行
* 为什么上述两个过程等同？[origin](https://www.bilibili.com/video/BV1ib411t7YR?p=10) 9: 30 左右
	* 考虑一个投影矩阵，比如把二维平面投影到一维向量，投影后的二维基一维坐标，就是一维基的二维坐标
	* 于是：1x2 矩阵和 2x1 向量乘积 = 点积 = 2x1 向量投影
* 毕竟，投影变换是线性的，必然可以用矩阵描述；又因为 1x2 矩阵和 2x1 向量乘积 & 点积的计算过程相同，所以会和某个二维向量有关
* 于是，与二维空间中某个点做点积，就和投影的变换对应了
* 可以不把向量看做箭头，而是看做特定空间变换（投影到一维）的一个载体
