* learn get started with Unity





There are many more learning resources on the Unity Learn platform!
-   Start with one of our Creator Kits: [RPG](https://learn.unity.com/project/creator-kit-rpg), [FPS](https://learn.unity.com/project/creator-kit-fps), or [Puzzle](https://learn.unity.com/project/creator-kit-puzzle)
-   [If you’re interested in coding, follow the Junior Programmer pathway.](https://learn.unity.com/pathway/junior-programmer) 
-   [Explore our live learning events.](https://learn.unity.com/live-calendar) 
-   [Browse hundreds of learning resources on learn.unity.com.](https://learn.unity.com/) 

Whatever your experience and interests, there is something on Unity Learn for you. Enjoy the journey!


In addition to the Learn Platform, the Unity Learning Ecosystem includes:

-   [Unity Learners Group](https://learn.unity.com/g/unity-learners): A dedicated message board for anyone learning Unity.
-   [Unity Microgames](https://blogs.unity3d.com/2020/09/09/create-your-first-game-brick-by-virtual-brick-with-the-lego-microgame/): Guided experiences designed to get new users working in the Editor quickly and easily, where you’ll follow tutorials embedded in the Unity Editor to better understand how everything works together, while making your own creative decisions and personalizing your game along the way.
-   [Learn tab in Unity Hub](https://docs.unity3d.com/hub/manual/Learn.html): gives you access to a variety of learning resources to help you get started with Unity.
-   [Unity Documentation](https://docs.unity3d.com/Manual/UnityManual.html): The Unity User Manual and Unity Scripting API Reference helps you learn how to use the Unity Editor and its associated services.
-   [Unity Answers](https://answers.unity3d.com/) or [Unity Forums](https://forum.unity3d.com/): here you can ask questions and search answers.
-   [Unity YouTube Channel](https://www.youtube.com/user/Unity3D) : the official Youtube channel with videos to learn more about Unity, the kinds of projects made with the Engine, tutorials, demos and more.
-   [Unity Blog](https://blogs.unity3d.com/): provides news, announcements and a glimpse inside Unity Technologies
-   [Unity “Tuesday Tips” on Twitter:](https://mobile.twitter.com/unitygames) Every Tuesday creators share their favorite [unitytips](https://twitter.com/hashtag/unitytips?src=hashtag_click)
-   [Asset Store help](https://unity3d.com/asset-store/help): help with Asset Store content sharing.
