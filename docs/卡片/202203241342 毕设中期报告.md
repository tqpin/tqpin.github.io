**课题进展情况（含已取得的阶段性成果）：---300字至1000字。**

我们进一步调查了优化和回归两种方法的论文发展，发现优化的方法在 SMPLify 之后很少有发展，只有在生成和预测下一帧的人体模型的任务中，对构建人体模型先验的方法有所改进。而因为数据集的扩大以及 3d 标注的增加，后续论文对其他约束项，如骨骼长度、旋转角度等，都倾向于直接利用回归的方式从图像中直接学习。

而神经网络的方法后续有很多的扩展和讨论，比如利用日益精准的 3d 骨骼预测直接从图像学习 SMPL 模型的两个参数，又利用 GCN 或者 transformer 直接学习 SMPL 模型的所有顶点。而在 2020 年的时候 Pose2Pose 意识到 SMPL 模型的旋转参数可以分解成扭转和旋转两方面，其中旋转可以直接利用 3D 的骨架信息得到，而扭转则更依赖于图像的视觉信息。于是在 2021 年的时候 HybrIK 利用这个分解，从图像中直接学习形状和扭转，然后从 3D 骨架中得到旋转，通过一个 IK 框架，最终得到 SMPL 需要的两个参数。 

另一方面，优化和网络回归两种结合的方法并不多见，除了 SPIN 提出的模型，我们并没有见到后续的发展。原因主要是优化和回归不能很紧密的结合在同一个学习流程中，并且优化约束的优势随着数据量的增加减弱了，而优化速度缓慢的劣势并没有很好的解决。

但我们了解到优化能够以更好的方式结合进回归的方法中，产生一个完整一体的学习过程。并且现在其实并没有一个完整的数据集方便我们学习和验证，我们需要把很多数据集拼凑在一起，就会产生一些域的差距，需要额外的技术减少这些差距。因此我们正在尝试把优化插入基于 HybrIK 的方法中。并且我们发现了 HybrIK 的 IK 部分还是依赖于传统的机器人学中的 IK 方法，并不完全适应于人体重建任务，可能并不会收敛于最优解。

总的来说，在理论方面我们想要更密切的结合优化和回归，并基于 HybrIK 进行改进，除了加入优化的部分，我们还找到了 HybrIK 的几个可以改进的地方，比如利用 3D 骨架预测结果帮助学习更准确的形状参数，以及发现了 IK 部分的一些问题；实验方面我们已经整理好了数据集和标注，跑通了原论文的代码，了解了代码的结构和插入优化部分的位置。


**存在的问题及解决思路： ---200字至1000字。**

IK 部分的方法修改：机器人中的 IK 和三维人体姿态估计中的 IK 存在一定区别，机器人中不需要考虑机械臂的长度误差，所以在求解 IK 的过程中假定长度是不变的，只需要考虑角度的变化。而三维人体姿态估计需要考虑骨架长度的不确定性，HybrIK 的解决方法是自适应迭代，但可能很难找到全局的最优解，并且迭代过程较慢。我们想要找到一个更适合姿态估计的方法，同时拟合所有的旋转和骨架长度，来替代 HybrIK 的 IK 部分。

优化部分的建模：我们想要在模型中加入优化的部分，利用回归部分缩小优化的搜索空间，以此加快优化的速度，并利用优化的部分使模型更容易的到达最佳值。我们可以把骨骼旋转角度、扭曲范围、姿势的先验信息加入优化部分，具体的数学语言还需要修改。

训练的数据集规模：优化的方法虽然可以使得模型收敛在更好的位置，但比回归的收敛速度慢很多，又因为训练和测试的数据集数量扩大，回归的效果也很不错了。我们想要在比较小的数据集上训练和测试，可以更加体现出优化方法的优势。


**下一步的工作计划和研究内容：---200字至1000字。**

1. 让原网络的 3D 骨架预测结果帮助全连接网络学习更准确的形状参数；
2. 更具体的讨论优化部分的插入方法，重新建模总体的目标函数；
3. 寻找更加适合动态或不确定度较大的机械臂的 IK 求解方式，尝试利用全连接层或者新的全局优化的方法替代原有的 IK 部分，输入旋转、扭转和形状参数，得到 SMPL 模型的两个参数；
4. 把数据集修改成较小的规模，让网络现在小数据集上训练，并对比加入优化和不加优化在小规模数据上的准确度；
5. 修改完代码，完成和原论文的对比实验，并写完终期报告；
