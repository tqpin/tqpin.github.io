研究背景与意义：

从单一图像估计 3D 人体模型一直以来吸引了广泛注意力，以往的工作大多集中在分别讨论优化和回归两种方法的优缺点。优化的方法试图在给定一个人体参数模型的情况下迭代拟合人体的姿势和形状，以最佳地解释 2D 观测，大多情况下就是 2D 的关节位置。因为优化方法显示地构建了模型，它通常能得到较好的拟合，但速度非常缓慢，并极其依赖初始位置。而回归的方法试图利用深度网络直接从图像中学习模型参数。该方法能够考虑所有的像素，但需要大量的数据进行训练，而获取 3D 姿势的准确监督值是非常昂贵的，我们的社区并没有很多的监督数据。我们希望可以结合两者优势，构建一个端到端的模型，把优化和回归的方法放在一个统一的框架下，发挥各自优势处理目标函数中的凸和非凸部分。相比于 SPIN，我们的模型使优化和回归两种方法结合更加紧密，预计能更好的彼此互补。 

主要内容（含技术指标）及目标：

我们的目标是构建一个结合优化和回归两种方法的端到端模型，用以从单一图像估计 3D 人体模型。而常用的 3D 人体姿态估计的评价指标有如下几种： 1. MPJPE 平均关节位置误差：测量估计值和真实值的平均欧氏距离。 2. PCK 正确关节百分比：当某关节和真实值的距离小于 50mm 时被认为是正确的，PCK 通常考虑肩膀，肘部，手腕，臀部，膝盖和脚踝几个关节。 3. 以及一些 MPJPE 的变种，如衡量角度估计的 MPJAE 和基于 Procrustes 分析的 MPJAE_PA。 

研究进度安排：

1. 阅读以往论文和代码，理解所有约束条件，构建一个目标函数，并分离出非凸部分（2021/12 - 2021/1） 3. 构建网络，寻找合适的优化框架容纳优化部分和回归部分（2022/1 - 2022/3） 4. 调试代码，在各个数据集上测试，并与现有模型比较（2022/3 - 2022/4） 5. 撰写报告（2022/4 - 2022/5）
