* 药化：（pharmaceutics ）化学结构，合成，分析，质量控制
* 药理：(pharmacology)是研究药物与机体(含病原体)相互作用及其规律和作用机制的一门学科。药理学主要指研究有关使用化学物质治疗疾病时引起机体机能变化机制的学问。药物的作用机理，药物在体内的受体，药物与受体结合的强度和时间过程。产生生物学效应的强度和时间过程。基本是动物实验或者细胞
* 药剂：注射剂和片剂产生的效应有和不同，药物的吸收，分布，代谢，排泄的过程。就是关于药物剂型像片剂胶囊啥的。是研究药物配制理论、生产技术以及质量控制合理利用等内容的综合性应用技术学科。其基本任务是研究将药物制成适宜的剂型，保证以质量优良的制剂满足医疗卫生工作的需要。
