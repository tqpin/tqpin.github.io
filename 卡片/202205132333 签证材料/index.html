    真实学生身份
    足够的资金
    一定会回国
    最重要的先说，先回答问题

    Good morning sir/madam

    为什么来美国？
    研究生的教育资源丰富；美国是一个非常多元化的国家，想 get along with 不同的 faculty from different country；美国的教育体系注重理论和实践相结合，practical experiments，有利于我之后回国找工作

    为什么 osu 为什么美国为什么 major，研究方向，software engineering

    和谁一起去？No just myself

    General staff in

    有个存款有六十万，还有房啊车啊 to support me

    我家就我一个人得回去养老，

    我准备
    - [ ] 学习规划/个人简历：的简历可以用申请时的简历。如果录取专业属于高风险专业，最好提供学习规划。
    - [ ] 各种证明自己优秀的证明（如作品集，获奖证书，发表的文章等）
    - [ ] 各种能够证明自己学生身份的附加材料（如学生证，高考凭证，学校餐卡，学校借书证等）

    家人准备
    - [ ] 亲属关系公证或户口本
    * 备注：这个材料是为了证明学生与资金资助人是有一定的亲属关系的，一般情况下出具户口本就可以，出生证明也可以。
    - [ ] 父母（资助人）在职收入证明中英文版本
    * 备注：即使是父母开公司的，最好也出具在职收入证明，可以让财务处或是人事部盖章，或盖公司总章也可以。用来证明自己的身份以及收入。记住，中文版本的要盖章，英文版本的也要盖章。在职收入证明最好用公司自己的抬头纸打印。如果父母不是在同一家公司，建议用不同版本的收入证明，以防签证官多疑。
    - [ ] 护照
    - [ ] 全家福、出国旅游，或是公司照片：备注：全家福只需你跟爸妈在一起拍的照片就可以。出国旅游照片是良好记录的体现。公司照片可以更直观证明公司的真实性。
    - [ ] 父母单位宣传册或公司介绍：这个是证明公司真实性以及让签证官了解学生父母公司情况的有力证据。

    需要打印
    - [ ] 美国签证照片
    - [ ] DS160 (非移民签证在线申请表) 确认页
    - [ ] 签证预约确认页：备注：CGI 号码和您的护照为您缴纳签证费时候的身份凭证;CGI 是负责美国签证申请系统的一家网络技术公司的名字, 无特殊含义。
    - [ ] I-20 表：需签名，签证面谈过后需向签证官要回来，谨记。
    - [ ] SEVIS FEE 缴费证明（学生和交流访问者信息系统缴费成功的收据）：备注：SEVIS 请提供信用卡信息。
    - [ ] 成绩单
    - [ ] 在读证明
    - [ ] 美方学校录取通知书：可带多份，最终决定去的学校录取通知书放在众多录取通知书的最上面。如果美方学校寄过来一些学校录取资料，也可以带着。但注意美方学校录取通知书上所写的收件地址是哪里？一般来说，写的是北京。如果是北京的学生，那么就没有问题。如果是北京以外的学生，那么宁可不带录取通知书，一旦带了，最好能解释清楚为什么在北京收这个邮件。
    - [ ] 美国大学课表：只要大致列出所申请大学专业的课程就可以。大学课表一般可以在学校官网上的 ACADEMIC 里找到。不必列出全部，可以列出部分，打印在一张纸上即可。
    - [ ] 各类考试成绩单（如雅思/托福/GRE/GMAT/SAT 成绩单等）：这些成绩单如果没有纸质版的，可以在网上查成绩，把成绩截屏出来，然后打印出来。
    - [ ] 父母（资助人）在职收入证明中英文版本

    资金类
    - [ ] 银行存单/存折/储蓄卡/存款证明由于在美完成本科需要 80 万到 100 万，申请时已提供 50 万存款证明，最好能再提供 30 万到 100 万：备注：这些是户头上的资金证明。资金证明的数额应以能够支持学生在美国学习以及生活的总消费为准。根据学生
    - [ ] 在国外学习及生活的费用，资金证明金额大约是在 80 万到 100 万不等（本科）。但请记住，覆盖学生学习及生活费用的数额是资金证明的最低标准。如果资金没有困难。建议可以提供超出最低标准的资金证明数额，来证明自己财力雄厚。一般 100 万以上就可以。若是提供存单和存折，最好是有较大数额进出账记录的那份。
    - [ ] 银行流水或对账单：备注：这是证明财产信用以及来源的资金材料，打印签证 3-6 个月前的流水或是对账单就可以。如果是有工作的，可以把工资卡的流水和对账单打印出来，证明自己收入的合法性以及来源，如果自己有公司的，那么可以把公司账户的流水或对账单打印出来，证明自己公司的利润以及自己的收益。
    - [ ] 基金/股票/大额保险等投资证明：备注：这些是补充性资金材料，如果有的话，可以提供，作为资金证明的补充，如果没有，则不需要提供。
    - [ ] 房产证：备注：作用不仅仅是资金证明，也是学生在国外学习会最终回来的经济牵挂证明。
    - [ ] 汽车证：备注：作用不仅仅是资金证明，也是学生在国外学习会最终回来的经济牵挂证明。
    - [ ] 营业执照：备注：如果不是自己公司，可以提供公司营业执照的复印件，复印件必须加盖公司红章才有效力。如果是自己公司，可以提供公司营业执照的原件，正本或副本都可以。不论是什么样的营业执照证明，都必须是经过年检，有年检章的证明。

    温馨提示：
    1. 申请者应尽可能多提供证明自己的材料，例如公司或教授提供的 supporting letter 或 email，来证明自己在国内的优秀表现或资金问题。
    2. 强调！！！！美国签证所用的材料必须全部是原件。如果无法提供原件，至少要提供盖过权威公章的复印材料。
    3. 签证材料准备的原则是：简明、精炼、重点突出。把签证材料一味地塞给签证官是错误的做法，应该签证官向你索要什么材料，你拿出对应的材料，边展示边向签证官解释。
    4. 重要性材料是非带不可的，补充性材料可以根据个人情况准备。资金证明材料也是很重要的材料，具体准备方法及标准可以参照我底下所写的备注信息。
